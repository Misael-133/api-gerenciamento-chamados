# API de Gerenciamento de Chamados

Projeto backend desenvolvido com Flask e SQLite simulando um sistema corporativo de abertura e gerenciamento de chamados técnicos.

## 🚀 Tecnologias utilizadas
- Python
- Flask
- SQLite
- API REST
- JSON

## 📌 Funcionalidades
- Criar chamados via API
- Listar chamados cadastrados
- Integração entre Flask e banco de dados SQLite
- Retorno de dados em formato JSON

## 🎯 Objetivo do projeto
Praticar conceitos de backend, APIs REST, manipulação de banco de dados e estruturação de sistemas reais utilizados em empresas.
